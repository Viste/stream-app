How to Stream

1. First we’re going to ask you to install a streaming software that is compatible with HLS. We’d recommend either OBS Studio or Ffmpeg which are both free.

2. During the set up, you need to connect the software to our HLS server address (rtmp://145.220.75.94:21204/live)

3. You will have to create a stream key, which will be sent to your viewers. This is what will give them access to your stream once it’s started.

4. Now just start your stream and it will be accessible on the “Join a Livestream” page!
